# tomek / web-ui

Web interface to control and monitor the line follower.

## Installation

This project requires [nvm](https://github.com/nvm-sh/nvm), a version manager for Node.js. If you donâ€™t have it installed, follow the instructions [here](https://github.com/nvm-sh/nvm#installing-and-updating).

1. Use the correct Node.js version:

   ```bash
   nvm use
   ```

   This command reads the Node.js version specified in the `.nvmrc` file and switches to it.

2. Install dependencies:
   ```bash
   npm install
   ```

## Usage

### Local development

```bash
npm start
```

### Build for production

```bash
npm run build
```
